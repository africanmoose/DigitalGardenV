- Code might be complex, but it should be easy to read and understand when complete.

### Details
- This might involve using [[1x - References/11 Notes/11.03 Ideas/Programming principles that work in real life - Use consistent standards\|Programming principles that work in real life - Use consistent standards]], or simply having clear documentation within the code
